# Funo para plotar os pontos e a reta de regresso
def plotar_pontos_regressao(X, eixo_x, y, eixo_y, y_pred):
    plt.figure(figsize = (12, 8))
    plt.xlabel(eixo_x, size = 16)
    plt.ylabel(eixo_y, size = 16)
    plt.plot(X[eixo_x], y, "o", label = "Dados Reais")
    plt.plot(X[eixo_x], y_pred, "r-", label = "Reta de Regresso (Previses do Modelo)")
    plt.legend(loc = "best")
    plt.show()

3. Conclusão¶

Pelo valor de R2 obtido, conclui-se que o modelo de regresso linear simples que usa a rea em m2 para prever o valor do aluguel usando este conjunto de dados no gerou resultados satisfatrios para resolver este problema de negcio.

A biblioteca Statsmodels se provou bem simples, eficiente e descritiva para treinar um modelo de regresso linear simples e facilitou a anlise estatstica do problema.
