    plt.xlabel(eixo_x, size = 16)
    plt.ylabel(eixo_y, size = 16)
    plt.plot(x, y, "o", label = "Dados Reais")

O modelo de regresso linear simples criado pode ser representado por:

$$Y=48.1819X+3596.7837$$

,onde Y o salrio, e X a quantidade de horas de estudo por ms.

3. Conclusão¶

Pelo valor obtido de 0.59 para R2, conclui-se que o modelo de regresso linear simples que usa a quantidade de horas de estudo por ms para prever o valor do salrio usando este conjunto de dados gerou resultados razoavelmente satisfatrios para resolver este problema de negcio.

A biblioteca Scikit-Learn, assim como o Statsmodels, se provou bem simples e eficiente para treinar um modelo de regresso linear simples, e mais recomendada para criao de modelos de machine learning que sero colocados em produo..
